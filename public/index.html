FAQ

คำถามที่พบบ่อย

ต้องมีความรู้ด้านวิดีโอหรือ AI ไหม?▼

ไม่ต้องเลยครับ! StoryPro ออกแบบให้ใช้งานง่ายมาก แค่อัพรูปสินค้า เลือกนายแบบ เลือกฉาก แล้วกดปุ่มเดียว ระบบจัดการให้หมด

ต้องใช้แพ็คเกจ Google Flow ไหนถึงจะใช้ได้?▼

ใช้ได้ทุกแพ็คเกจของ Google Flow เลยครับ ระบบใช้เครดิตจาก Google Flow ที่มีอยู่แล้ว ไม่ต้องอัพเกรดพิเศษ สมัครได้ที่ labs.google/flow

รับลิงก์นานแค่ไหนหลังโอนเงิน?▼

แอดมิน approve ภายใน 24 ชั่วโมง ส่วนใหญ่เร็วกว่านั้น พอ approve แล้วลิงก์จะส่งไปทางอีเมลทันที

ใช้ได้กี่สินค้า?▼

ใช้ได้ไม่จำกัดสินค้าครับ จ่ายครั้งเดียวใช้ได้ตลอด ทำคลิปได้ทุกสินค้า

มีปัญหาติดต่อได้ทางไหน?▼

ติดต่อได้ทางอีเมล boategrshop@gmail.com ยินดีช่วยเสมอครับ

ใช้งานยังไง ติดตั้งโปรแกรมยากไหม?▼

เข้าใช้งานที่ Tools บนเว็บไซต์ Google Flow หลังจากกดลิงก์โปรแกรมแล้วได้ทันที ไม่ต้องติดตั้งอะไรเพิ่ม

ต้องมี Gemini Ultra หรือ Pro ไหม?▼

ใช้ได้ทั้งสองอย่าง และตัวฟรีก็ใช้ได้ครับ แต่เครดิตจะไม่เท่ากัน ทุกๆ คลิปใช้เครดิตทำ

349 บาท จ่ายครั้งเดียวไหม?▼

จ่ายครั้งเดียวซื้อโปรแกรมแบบถาวร พร้อมการอัพเดท ส่วนรายเดือนจ่ายค่าเครดิตกับ Google Flow เอง

ทำอะไรได้บ้าง?▼

ทำคลิปขายของ ปักตะกร้าหลากหลายแนว คลิปเล่าเรื่อง คลิปละคร ทำได้ถึง 30 ฉาก

เจนโหมด 0 เครดิตได้ไหม?▼

ปัจจุบัน Google ไม่อนุญาตให้ใช้ 0 เครดิตนอกแพลตฟอร์ม ทั้ง API หรืออื่นๆ ต้องใช้หน้าแพลตฟอร์ม Google Flow เท่านั้น

ใช้บอทไหม?▼

โปรแกรมไม่ใช่บอทครับ เป็นโปรแกรมสำเร็จรูปที่คั่นกลางระหว่างผู้ใช้กับ Flow ทำให้ทำงานง่ายขึ้น สะดวกขึ้น

มีไลน์กลุ่ม สามารถสอบถามได้ครับ แต่ถ้าใครก่อดราม่า เตะออกทุกกรณี

ใช้ API ไหม?▼

ไม่ใช้ API ครับ เพราะฉะนั้นไม่มีค่าใช้จ่ายอื่นๆ อีก นอกจากค่าเครดิต Flow

ทำในคอมหรือมือถือได้ไหม?▼

ทำได้ทั้งคอมและมือถือ หรือแท็บเล็ตแอนดรอยด์ครับ iOS ปัจจุบันยังไม่ได้ จะมีการอัพเดทเร็วๆ นี้แน่นอน
